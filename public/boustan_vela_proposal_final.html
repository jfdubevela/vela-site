01  ·  The Opportunity
From data silos
to one source of truth.
Boustan operates across dozens of franchise locations, each generating valuable data through its POS, delivery platforms, and marketing channels. Today, that data lives in separate systems that don't talk to each other. The opportunity is to change that, and turn raw transaction data into a real-time intelligence layer that drives better decisions at every level of the organization.
Starting Point
Internal groundwork has already identified the tools in play: MYR POS (current system, with AUPHAM migration planned for September), Uber Eats, DoorDash, U-Eat for delivery, HubSpot for CRM, and Google & Meta for paid media. The destination: Power BI as the central analytics layer, with automated task creation in Monday.com when the data signals action.

Phase 02  ·  Pilot Implementation (5–10 Franchisees) Timeline TBD after Phase 1
Build It Right. Prove It Works.
Everything in this project happens centrally. The pipelines, dashboards, and automations all live in Boustan's platform stack. Franchisees continue using their POS and delivery apps exactly as before. Nothing is installed or changed at the restaurant level. The pilot starts with 5–10 franchisees' data feeds to validate that the architecture, the dashboard logic, and the Monday.com automations all work as expected at a manageable scale before connecting the full network.
What gets delivered
Automated data pipelines connecting all sources, MYR, Uber Eats, DoorDash, U-Eat, HubSpot, Google Ads and Meta, into a unified Power BI data model via Power Automate
Power BI dashboards tailored to Boustan's operations, sales by location, delivery performance, marketing ROI, and cross-source trends
Monday.com as the operations layer, surfacing Power BI data directly where your team already works, not as a passive report viewer but as an active command center
What Monday.com can do with the data
Embedded Power BI reports — Live dashboards displayed directly inside a Monday board. Your ops team sees sales and delivery performance without ever leaving Monday.
Automated status updates — Each location's item in Monday switches automatically between green, yellow, and red based on real-time thresholds: sales vs. target, delivery rating, complaint volume.
Automatic task creation — When a signal appears in the data (sales below target for 3 consecutive days, a delivery spike, a rating drop), a task is created in Monday, assigned to the right person, with the relevant data already attached.
Franchisee KPI scorecards — Each location has a Monday item that always shows its current week's numbers: revenue, order volume, delivery rating, marketing spend. One board, 45 locations, always up to date.
Automated weekly digest — Every Monday morning, a summary update is posted automatically in the relevant Monday thread with key highlights for each region manager, pulled directly from Power BI.
Threshold alerts — When any metric crosses a predefined limit (e.g., ad spend with no corresponding sales lift, a location's volume suddenly halved), Monday sends a notification to the right stakeholder before anyone notices manually.
Architecture built for portability, designed from day one to migrate cleanly to AUPHAM in September with minimal rework
Team training and full system documentation, so Boustan owns and understands the system, not just uses it
Phase 03  ·  Full Network Coverage After pilot validation
Connect the Rest of the Network.
The architecture is built. The pipelines work. The dashboards are live. Phase 3 is about extending that same system to the remaining 35–40 franchisees. Since all franchisees already send data in the same format, this is an expansion, not a rebuild. The work happens entirely at the platform level: no changes at the restaurant level, no interaction required from franchisees.
What gets done
Remaining franchisees' data feeds added to the existing pipelines. Since the format is already uniform across the network, each addition follows the same pattern established in the pilot
Dashboards and Monday.com boards validated at full network scale, all 45 franchisees visible, segmented, and trackable in a single view
AUPHAM migration: the data architecture transitions to the new POS system when Boustan switches in September, with minimal rework required thanks to the portability built in from day one
Training for Boustan's management and operations team on reading the dashboards, interpreting the data, and acting on Monday.com alerts and reports
